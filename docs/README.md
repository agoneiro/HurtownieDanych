# Dokumentacja

Folder `docs` zawiera dokumentację projektową dotyczącą analizy, wymagań oraz projektu hurtowni danych. Dokumenty te stanowią podstawę zrozumienia procesów biznesowych, zakresu wymagań oraz finalnej architektury proponowanego rozwiązania.

---

## Zawartość folderu

W folderze znajdują się następujące dokumenty (w formacie PDF):
- `ProcessesSpecification.pdf` - opisujący procesy biznesowe podlegające analizie przez projektowaną hurtownię danych

- `RequirementsProcessSpecification.pdf` - określający wymagania dla projektowanej hurtowni danych

- `ProjectReport.pdf` - kompletny projekt hurtowni danych
